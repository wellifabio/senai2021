- Instalar o NodeJS (https://nodejs.org/en/)
- Verificar instalação utilizando o terminal (CMD):

	* node -v
	* npm -v
	
- Instalar o expo utilizando o terminal (CMD) :

	* npm install -g expo-cli
	
- Ao criar um projeto, será criada uma nova pasta com o nome do mesmo automaticamente.

- Para criar um novo projeto é necessário acessar o prompt de comando pelo CMD ou VSCODE 
 e executar o seguinte comando:
 
	* expo init nome_do_projeto
	
- Ao executar o comando acima será listada opções de template do projeto, selecione uma 
 de acordo com sua necessidade (Para as aulas iniciais selecione a primeira opção (Blank))
 
- Criado o projeto, acesse sua pasta no prompt:

	* cd nome_do_projeto
	
- Para compilar e carregar seu projeto, utilize o comando:

	* expo start
	
- Feito isso, será carregado uma aba no navegador com as opções de execução do seu projeto,
 permitindo a utilização do navegador, emuladores e seu próprio dispositivo mobile.
 (Para as aulas iniciais estamos utilizando a opção 'Run in web browser')
