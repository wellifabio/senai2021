Observar  que  estão  sendo  filtrados  somente  os  arquivos  JSPs.
Na  requisição  foi  criada uma lista de IPs, recuperou-se a data atual, criou-se um objeto para passar os valores e o atributo mensagem, que inicialmente está com o valor “acesso negado”. Por fim, foi criada uma estrutura condicional para verificar se o IP do usuário é o mesmo da aplicação – caso contrário, verificar se o IP do usuário está na lista dos que terão acesso ao sistema. Caso retorne verdadeiro, adiciona-se o setAttributepara enviar para tela com o objeto. Depois do chain.doFilter foi usado o dispatcherpara redirecionar para o arquivo status.jsp
